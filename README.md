
# NewTabPage
a fork of [HumbleNewTabPage](https://github.com/quodroc/HumbleNewTabPage)  
Redesigned new tab page featuring your bookmarks, most visited

## Features
* Simple, clean design
* Fast loading and lightweight
* Options (Fonts, Colors, Content, and more)
* Reordering Folders
* Adaptive Layout
* TopSites-BlackList
* Icons Per Row
